# Sleman Slowpitch Reunion Invitation

A fun, interactive invitation for the Sleman Slowpitch Reunion Call.

## Features
- Interactive stickers and emojis.
- Countdown timer to the reunion.
- RSVP system with confetti effects.
- Secret click counter easter egg.

## Development
This is a static HTML project. Simply open `index.html` in your browser to view the invitation.
